Julia language bar and character compositions

This provides two ways to easily enter Julia characters into pretty much any input field on any webpage:

  1. Enter a combo sequence and hit Tab, e.g. \piTab gives π. See the complete list of completions.
  2. Click on symbols in the language bar.

Try it here:

Hover over a symbol in the language bar for combo sequences.

Installation

Desktop browser

  1. Drag this link to your bookmarks bar: Julia

Mobile browser

  1. Add a bookmark for this page.
  2. Edit the bookmark's target URL address to javascript:(d=>{let%20e=d.createElement("script");e.src="https://abrudz.github.io/lb/julia.js";d.body.appendChild(e)})(document)

Webpage

  1. Include this HTML in your page: <script src="https://abrudz.github.io/lb/julia.js"></script>

Activation

  1. Navigate to the web page where you want to enter Julia characters.
  2. Click on the Julia bookmark. The language bar should appear at the web page's top.
